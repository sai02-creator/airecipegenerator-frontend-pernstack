## 🌟 Overview

The **AI Recipe Generator Frontend** is a modern React application that allows users to generate recipes using AI, manage pantry items, and plan meals.

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 🤖 AI Recipe Generation
* 🥫 Pantry Management
* 📅 Meal Planning
* 🛒 Shopping List

---

## 🛠 Tech Stack

* React (Vite)
* Axios
* CSS
* AWS S3 (Hosting)

---

## 🌐 Live Demo

👉 http://ai-recipe-generator-frontend-sai123.s3-website-ap-southeast-2.amazonaws.com

## ⚙️ Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://<EC2-IP>:8000/api
```

## 💻 Run Locally

```
npm install
npm run dev
```

## 📦 Build

```
npm run build
```
